Project Overview

This is a college project that demonstrates an automated attendance marking on a predifined time slots for every course. It uses tensorflow ,MTCNN, FaceNet to implement live face recognition.

How It Works

The application allows lectures to set the timing of the classes so that when the application at backend should start taking attendance.

Feature1: Setting class times so that lectures can have flexibility of taking attendance on desired time.

Feature2: Lecturers will also have email notification so that if time slots of attendance marking get overriden by another lecturers.

Feature3: Studnets facial data wil be collected at admission process.

Feature4: After attendence marking students will recieve an email of absent status and if they got false messege they can reslove with lecturers.

Technologies Used

This project was built using React, Node.js, Express, MongoDB to provide dashborad for lecturers and tensorflow ,MTCNN, FaceNet to implement live face recognition.
